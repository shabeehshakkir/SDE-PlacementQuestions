# String

### What is string ?<br>
- Defined are defined as an array of characters
- The string is terminated with a special character '\0'
#### C++
```cpp
    string s = "hello";
    cout<<s.find("lo");
    cout<<s.substr(1,2);
    // 1=>start index
    // 2=>length
```
#### Java
```java
    String s = "hello"; 
    System.out.println(indexOf("lo"));
    System.out.println(substring(1,2));
    // 1=>start index
    // 2=>end index (exclusive)
```
